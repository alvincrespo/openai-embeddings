# openai-embeddings

A Jupyter-based project (optimized for VS Code) for experimenting with OpenAI embeddings.

## Setup

1. Open this folder in VS Code.
2. Create/enable the virtual environment if not already active:
   - macOS/Linux: `source .venv/bin/activate`
3. Install dependencies:
   - `pip install jupyter ipykernel openai`
4. In VS Code, select the `.venv` interpreter for notebooks (Command Palette â†’ "Python: Select Interpreter").

## Usage

- Use the `notebooks/` folder for your experiments.
